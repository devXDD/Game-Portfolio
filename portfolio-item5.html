Low-poly Open world game

Made in Unity

Disclaimer: The project is only for educational purpose only. My intentions are not to release any copyrighted assests.

This project utilizes assets by Polygon and animation from Mixamo, to create an exiting open world experience.

Character Controller

Unity's input system has been used to handle all the user inputs. Along with this, animations like walking, sprinting, jumping and aiming have been imported from Mixamo and handled inside Unity using animation controller and scripts. Ray cast has also been implemented to introduce the shooting physics inside the game.

Level Design

RoadArchitect by MicroGSD has been used to create dynamic roads with subtle curves, which adds more open world feel to the game.

Features to be implemented

Unity's NAV mesh will be implemented to handle the NPCs in the game. Crouch and ledge grabbing will be also be introduced for more interactive behaviour of the Character.
